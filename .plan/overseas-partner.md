# Section 4：なぜ実装パートナーなのか

## 販売だけでは、日本市場で成功できません

比较表：

| 一般代理店 | 出雲創安   |
| ----- | ------ |
| 製品紹介  | 技術理解   |
| 営業中心  | 現場中心   |
| 契約後終了 | 運用まで支援 |
| 顧客紹介  | 導入成功支援 |
| 販売重視  | 長期運用重視 |

结论：

日本市場では、
「売ること」よりも「定着させること」が重要です。
